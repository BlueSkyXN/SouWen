# 无 API Key 场景实测报告

> ⚠️ **说明**：本报告基于 v0.3.0 实测数据，仅供参考。当前版本为 v0.6.3，数据源可用性可能因反爬策略变化而不同。

> **测试环境**: Hugging Face Spaces (免费 CPU 实例)
> **SouWen 版本**: v0.3.0
> **测试时间**: 2026-04-14
> **HTTP 后端**: curl_cffi (auto)

## 概述

SouWen 的设计理念是**零配置可用** — 尽可能多的数据源无需 API Key 即可工作。本文档记录了在无任何 API Key 的 HF Spaces 环境下，各数据源的真实表现。

---

## 论文搜索

所有免费论文源均为公开 API，无需 Key。

| 数据源 | WARP 关 | WARP 开 | 响应时间 | 说明 |
|--------|---------|---------|----------|------|
| OpenAlex | ✅ 正常 | ✅ 正常 | ~2.5s | 最稳定的学术源，覆盖 2.5 亿篇文献 |
| arXiv | ✅ 正常 | ✅ 正常 | ~2.3s | 预印本，计算机/物理/数学为主 |
| CrossRef | ✅ 正常 | ✅ 正常 | ~2.6s | DOI 元数据，覆盖面广 |
| PubMed | ✅ 正常 | ✅ 正常 | ~2.1s | 生物医学文献 |
| DBLP | ❌ 无结果 | ✅ 正常 | ~2.9s | 计算机科学，WARP 后恢复 |
| Semantic Scholar | ❌ 无结果 | ✅ 正常 | ~2.5s | AI 驱动的学术搜索，WARP 后恢复 |

**结论**: WARP 关闭时 4/6 可用，开启后 **6/6 全部可用**。

---

## 专利搜索

免费专利源较少，且大部分需要 API Key。

| 数据源 | WARP 关 | WARP 开 | 说明 |
|--------|---------|---------|------|
| Google Patents | ❌ 无结果 | ❌ 无结果 | 反爬严格，需要更高级的指纹伪装 |
| PatentsView | ❌ 无结果 | ❌ 无结果 | USPTO API 已迁移，需更新端点 |
| PQAI | ❌ 无结果 | ❌ 无结果 | 返回 401，免费额度可能已用尽 |

**结论**: 无 Key 场景下专利搜索暂不可用。

## 网页搜索

网页搜索引擎分为**爬虫引擎**（HTML 解析）和 **API 引擎**（需要 Key）。

### 爬虫引擎（无需 Key）

| 引擎 | WARP 关 | WARP 开 | 响应时间 | 说明 |
|------|---------|---------|----------|------|
| DuckDuckGo | ✅ 5条 | ✅ 5条 | ~4.0s | 最稳定，推荐默认引擎 |
| Bing | ✅ 5条 | ✅ 5条 | ~5.8s | 稳定可靠 |
| Yahoo | ⚠️ 不稳定 | ✅ 5条 | ~4.8s | 偶尔超时 |
| Baidu | ✅ 5条 | ✅ 5条 | ~6.5s | 中文搜索表现好 |
| Mojeek | ❌ 无结果 | ✅ 5条 | ~5.0s | WARP 后可用 |
| Google | ❌ 无结果 | ❌ 无结果 | ~17s | 反爬极强，即使 WARP 也无法绕过 |
| Startpage | ❌ 无结果 | ❌ 无结果 | ~4.4s | 使用 Google 后端，同样被封 |
| Brave | ❌ 无结果 | ❌ 无结果 | ~17s | 反爬较强 |
| Yandex | ❌ 无结果 | ❌ 无结果 | ~7.4s | 俄罗斯搜索引擎，地域限制 |
| Qwant | ❌ 无结果 | ❌ 无结果 | ~1.9s | 法国搜索引擎，API 变更 |
| Sogou | ❌ 无结果 | ❌ 无结果 | ~1.9s | 需要验证码 |
| Ecosia | ❌ 无结果 | ❌ 无结果 | ~2.0s | 使用 Bing 后端但限制更严 |
| Wikipedia | ❌ 无结果 | ❌ 无结果 | ~1.9s | 需检查解析逻辑 |

**结论**: WARP 关闭时 3-4 个引擎可用，开启后 **5 个引擎稳定可用**。

## WARP 代理效果

| 指标 | WARP 关 | WARP 开 | 提升 |
|------|---------|---------|------|
| 论文源可用数 | 4/6 | **6/6** | +2 |
| 网页引擎可用数 | 3-4/13 | **5/13** | +1-2 |
| 专利源可用数 | 0/3 | 0/3 | — |

WARP 对被 IP 限制的源（DBLP、Semantic Scholar、Mojeek）效果显著。对使用高级反爬技术的源（Google、Startpage、Brave）无效。

### 启用 WARP

```bash
# API 方式
curl -X POST http://localhost:8000/api/v1/admin/warp/enable

# 或在 Panel 面板中点击 WARP 开关
```

## 失败源分析

| 源 | 失败原因 | 可能的解决方案 |
|----|----------|----------------|
| Google (web) | 反爬检测（验证码+JS Challenge） | 暂不支持 |
| Startpage | Google 后端同样被封 | 暂不支持 |
| Brave (web) | 反爬检测 | 暂不支持 |
| Google Patents | 反爬 + 页面结构变化 | 暂不支持 |
| PatentsView | API 端点已迁移 | 待更新代码 |
| PQAI | 免费额度耗尽 | 等待额度恢复 |
| Yandex | 地域限制 + 反爬 | 使用俄罗斯 IP 代理 |
| Sogou | 强制验证码 | 暂不支持 |
