# src/souwen/server navigation card

Type: Guardrail card.
This directory is the FastAPI server, auth/middleware/limiter boundary, WARP support and embedded panel boundary.
Read `app.py`, `auth.py`, `middleware.py`, `limiter.py`, `warp.py`, `docs/api-reference.md`, and matching `tests/test_server/` files first.
Read this card for API lifecycle, auth, middleware, route registration, server state, WARP or panel artifact behavior.

## Why this is high-risk

- Auth role semantics protect Guest/User/Admin behavior.
- Server responses may expose config, plugin state or credentials if redaction regresses.
- `src/souwen/server/panel.html` is generated by the panel build and packaged into the wheel.

## Required before changes

- Identify whether the change belongs in app wiring, auth/middleware, route handlers or schemas.
- Read deeper cards for `routes/`, `routes/admin/` or `schemas/` when touching those directories.
- For panel artifact changes, modify `panel/` source and regenerate with `cd panel && npm run build:local`.

## Do not

- Do not leak passwords, tokens, cookies or API keys from config/admin responses.
- Do not default-open admin access; no-password admin requires explicit `SOUWEN_ADMIN_OPEN=1`.
- Do not perform slow network probes during server import or startup.
- Do not hand edit `src/souwen/server/panel.html`.

## Validation

- `pytest tests/test_server -v --tb=short`
- Broader server profile: `python scripts/ci/run_profile.py --profile server --profile minimal`
