# src/souwen/server navigation card

This directory is the FastAPI server and embedded panel boundary.
Read `app.py`, `auth.py`, `middleware.py`, `docs/api-reference.md`, and `tests/test_server/` first.
Read this card for API app lifecycle, auth, middleware, routes, schemas, WARP or panel artifact changes.

## Local invariants

- Guest/User/Admin role semantics must stay explicit.
- No-password admin access requires `SOUWEN_ADMIN_OPEN=1`.
- `/health` and `/readiness` must not depend on real external services.
- `src/souwen/server/panel.html` is generated by `cd panel && npm run build:local`.

## Do not

- Do not leak passwords, tokens, cookies or API keys from config/admin responses.
- Do not perform slow network probes during server import or startup.

## Validation

- `pytest tests/test_server -v --tb=short`
- `python scripts/ci/run_profile.py --profile server --profile minimal`
